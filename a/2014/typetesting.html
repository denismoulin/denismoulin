Typetesting

Experiment, Ux, Prototyping

I noticed that type testers in Foundry shops often offered the same experience, so I played with it a bit. It’s an endless subject, I guess, so I’ll keep updating this post. You can subscribe to updates here (no spam).

Attempting type testing is trying to find a new way to represent a subject that is dear to me, the font. It’s considering how to manipulate characters, images, and colors while maintaining functionality as a priority and driving toward the purchase/discovery of a font.

I’ve learned through experience that a font is a tremendous object that requires a huge amount of work. Let’s reconsider how we use and present them in order to showcase this labor.

I will continue to experiment with this project. If you have suggestions, don’t hesitate to use the download link or to send me your ideas.

This project is not done yet, I'll keep updating/add screens.
